## ✨ Overview

**NurDouha Pro** is a premium digital Quran platform that provides:

- 📖 Beautiful Arabic Quran text
- 🔊 High-quality audio recitation per verse & Surah
- 📚 Tafsir (interpretation) integration
- 🌍 English translation support
- ⚡ Fast, modern, responsive UI
- 🎨 Luxury spiritual design system



---

## 🚀 Tech Stack

- ⚛️ React (Vite)
- 🎨 Tailwind CSS
- 📡 REST APIs (Quran data sources)
- 🔊 HTML5 Audio Engine
- 🧠 Modular architecture (hooks + services)
- 📱 Fully responsive UI

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/abdellahaarab/NurDouhaPro.git
cd NurDouhaPro
````

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```

## 🌐 API Sources

This project uses open Quran APIs:

* Quran Surah Data
* Juz Index API
* Tafsir API
* English Translation API
* Audio Recitation MP3 streams

All sources are abstracted inside `/config/sources.js` for easy replacement.
